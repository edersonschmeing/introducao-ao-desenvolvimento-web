Lista ordenada <ol>

Uma ordered list, ou lista ordenada, é exibida com elementos numerados. Cada elemento deve ser marcado com a tag <li> (list item).

Exemplo

  1. Técnicos
  2. Graduação
  3. Pós-Graduação
  4. Mestrado
  5. Doutorado

Lista não-ordenada <ul>

A tag unordered list, ou lista não ordenada, mostra os elementos como tópicos. Cada elemento deve ser marcado com a tag <li> (list item).

Exemplo

  • Técnicos
  • Graduação
  • Pós-Graduação
  • Mestrado
  • Doutorado
